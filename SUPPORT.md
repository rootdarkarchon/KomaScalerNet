# Support status

KomaScaler.Net is an unsupported personal project provided as-is. No support channel, installation help, compatibility commitment, roadmap, or service-level expectation is offered.

GitHub Issues are intentionally disabled. Pull requests and feature requests are not guaranteed review. Discussions, email, and other individual support routes are not offered.

Problems with the upstream MangaJaNai models belong to the [MangaJaNai project](https://github.com/the-database/MangaJaNai), subject to that project's own support policy. NVIDIA driver, CUDA, cuDNN, TensorRT, and other vendor installation problems are outside this project's support scope.
